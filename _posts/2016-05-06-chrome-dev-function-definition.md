Suy nghĩ mãi chả biết viết cái gì, hôm này cũng không biết có gì mới, như hôm qua là ngon rồi :D.
May mà có cái nhỏ nhỏ mới để ý.
Ai dev web cũng biết cái Chrome Dev nó khỏe cở nào, nhưng ngặt nổi khi thấy một function nào đó thì ko có cái hotkey nào bẩm để chỉ tới function đó, đọc code rất khó chịu.

Bạn nào có cách nào hay chỉ nha, không thì xài cách của mình.
1. Tô đen function -> right click chọn `Add to watch`
![alt text](https://s3-ap-southeast-1.amazonaws.com/kipalog.com/h1.png_isarsjcne4)

2.Right click vào chữ function -> Chọn `Show function definition`
![alt text](https://s3-ap-southeast-1.amazonaws.com/kipalog.com/h1.png_dxh3nngpke)

Nhảy phát tới function đó lun, mấy cái file to to làm vậy chứ search mỏi tay lắm :).
